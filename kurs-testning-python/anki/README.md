# Anki flashcards

Jag laddade upp allt kursmaterial till ChatGPT och bad den skapa en Anki-fil med frågor på materialet.

På [ankiweb.net](https://apps.ankiweb.net) kan du läsa mer om Anki och ladda ned kostnadsfria klienter för Windows, macOS, Linux, iOS och Android. Det finns fler klienter för Anki-formatet (som är öppet), så det är bara att söka efter vilken som blir din favorit. På [ankiweb.net](https://ankiweb.net/about) finns det också en fri webbklient, så du kan köra allt i din webbläsare.

När du har en klient laddar du in `testing-python.apkg` filen i klienten och sätter igång och tränar.
