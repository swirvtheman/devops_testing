# Övning 10 - FizzBuzz med TDD

## Regler

- Givet talet n, returnera en sträng
- Om n är delbart med 3 -> "Fizz"
- Om n är delbart med 5 -> "Buzz"
- Om n är delbart med 3 och 5 -> "FizzBuzz"
- Annars -> returnera talet som en sträng (t.ex "1", "2")

## Viktigt

- Följ TDD-cykeln! Skriv ett test i taget.
- Kör testet. Se att det misslyckas (FAIL).
- Implementera. Kör testet. Se att det lyckas (PASS).
- Refaktorisera vid behov.
